Project1: Distributed Key-Value storage system

  • Engineered a distributed Key-Value storage system with high availability and strong consistency, utilizing the Raft consensus algorithm for operations including leader election and log replication.
  • Implemented a Multi-Raft architecture in Java to shared data effectively, enabling migration across multiple Raft Groups to achieve optimal load distribution and resilience.
  • Designed a Key-Value storage interface that ensures linearizable read and write operations, achieving stringent consistency levels required for critical applications.
  • Incorporated asynchronous Apply to increase the throughput and scalability, utilized ReadIndex and FollowerRead to reduce the latency of read operations by 42%, and implemented Prevote strategies to eliminate unnecessary elections to improve the system stability.
  • Employed MVCC for robust multi-version concurrency control and built on the Percolator model, a distributed database based on BigTable, to support efficient distributed transactions.

Project2: High-Performance Distributed Message Queue System

  • Engineered an efficient message storage mechanism comprising advanced data structures and persistent storage solutions, ensuring reliable message transfer between producers and consumers.
